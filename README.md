# Video Downloader    GIMME VIDEO
A simple GUI application for downloading videos from YouTube and other platforms.

## Features
- Easy-to-use graphical interface
- Quality selection
- Audio-only, video-only, or both options
- Live download progress with speed and ETA
- Animated status indicators

## Requirements
- Python 3.7 or higher
- yt-dlp library
- Pillow library (for displaying animated GIFs)
- ffmpeg 

## Installation / Visual studio version with full code 

    git clone https://github.com/ASusN1/ExtractorV2.git

## Installation - Ready to use immediately

    https://github.com/ASusN1/ExtractorV2/releases/tag/v1.0.7

## Usage

Run the application:

    python Main.py

Steps:
1. Paste video URL
2. Choose save location
3. Click "Get Video Info" to see quality options (optional)
4. Select download type (audio/video/both)
5. Click "Download Video"
